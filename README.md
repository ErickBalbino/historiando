# Historiando
 Site principal do historiando, um site educacional focado em história.
 
 Feito com:
* HTML
* CSS
* JAVASCRIPT

 Contém:
* Conteúdos mais importantes de história.
* Questões dinâmicas para praticar.
* Interface fácil e agradável.
